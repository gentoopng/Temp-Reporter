# Temp Reporter

🇯🇵[**日本語**](README.md) 👈ｲﾏｺｺ<br>
🇺🇸[**English**](docs/README_en.md)

## これは何

Arduinoを使って温湿度センサーの値を取得し、その内容をTwitterに投稿するものです。<br>
大学の授業の一環で作成しました。

## 使用言語・環境など
使用言語
- Python 3
- Arduino

環境（開発および動作確認）
- macOS Catalina (10.15.5)
- Arduino UNO
- Grove
    - DHT22 Temperature & Humidity Sensor Pro (AM2302)
    - (LCD RGB Backlight)
